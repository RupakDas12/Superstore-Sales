# Superstore Sales Analysis

EDA on 9,994 order records to uncover revenue trends, top products, and regional insights.

## Overview

| Detail | Info |
| --- | --- |
| Dataset | Sample Superstore (Kaggle) |
| Records | 9,994 orders |
| Period | 4 years |
| Tools | Python, Pandas, Matplotlib, Seaborn, Jupyter |

## Key Findings

- West region is the top revenue generator (~$25K)
- Technology is the highest-grossing category (~$36K)
- November-December are peak sales months every year
- Canon imageCLASS 2200 Copier leads at $1.6K (2.5x next product)
